## Wordle Game
A simple wordle game built in React. 

## How to Play
- The game is played in 6 attempts.
- Each attempt is a row of 5 squares.
- The squares turn green if the letter is in the correct position.
- The squares turn yellow if the letter is in the word but not in the correct position.
- The squares turn gray if the letter is not in the word.
- The game is won if the word is guessed in 6 attempts.
- The game is lost if the word is not guessed in 6 attempts.
